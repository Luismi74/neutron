####First shot :heart_eyes:

[![NeutronKit Logo](http://neutronkit.com/img/screenshot-1.png)](http://neutronkit.com)

###Development

####Install node dependencies

```
npm install
```

####Open the example app

```
gulp serve && gulp serve:electron
```

####Modifying & Watch Sass files

```
gulp serve
```
